* #### css

```css
h1{
    text-align: center;
}
#container{
    margin:100px 170px;
}
.pictures{
    width:300px;
    height:200px;
}
#photo1{
    background-color:rgb(43, 41, 41);
}
#photo2{
        background-color:gray;
}
#photo3{
        background-color:rgb(64, 58, 92);
    }
.caption{
    position:absolute;
    top:100px;
    width:300px;
    height:200px;
    padding-top:20px;
    background-color:rgba(158,148,148,0,.815);
    opacity:0; /*화면상 안보이게 하는 설정 값 */
    transition:all 0.6s ease-in-out;
    z-index:10;
}
li{
    float:left;
    padding:0;
    margin:10px;
    overflow:hidden;
    position:relative;
    list-style:none;
}
li:hover .caption{
    opacity:1;
    transform:translateY(-100px);
}
.caption h2, .caption p{
    color: white;
    text-align: center;
}
```
